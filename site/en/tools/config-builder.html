MCP Config Snippet Builder

选择你要使用的 MCP Server，自动生成配置 JSON 片段。生成结果在浏览器本地处理，不上传任何信息。

选择Clients

选择 MCP Server

生成结果

请选择至少一个 MCP Server

生成的配置中，环境变量占位符（如 <your-token>）Required替换为你的真实值。不要把真实 Key 提交到公开仓库。

配置文件位置

Claude Desktop：

  • macOS：~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows：%APPDATA%\Claude\claude_desktop_config.json

使用说明

  1. 选择你使用的Clients。
  2. 勾选Required的 MCP Server。
  3. Copy生成的 JSON 配置。
  4. 粘贴到对应Clients的配置文件中。
  5. 替换环境变量占位符为真实值。
  6. 重启Clients使配置生效。

Important Notes

  • 本工具在浏览器本地生成配置，不收集任何信息。
  • 生成的配置仅供参考，实际格式以Clients官方文档为准。
