VS Code / Cline MCP 配置Guides

VS Code 中通过 Cline 扩展使用 MCP

配置文件位置

  • global：Cline 扩展设置面板中配置

配置步骤

  1. 找到或创建配置文件（路径见上方）。
  2. 在文件中添加 mcpServers 字段。
  3. 填入 MCP Server 的 command、args 和 env。
  4. 在 Cline 面板中点击刷新或重启扩展
  5. 在 Cline 对话中确认工具列表包含已配置的 MCP Server

Configuration Example

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/you/Documents"
      ]
    }
  }
}

FAQ

配置文件不存在怎么办？
手动创建该文件，内容为 {"mcpServers": {}}，然后添加你Required的 Server。
修改配置后不生效？
在 Cline 面板中点击刷新或重启扩展
怎么确认 MCP 已连接？
在 Cline 对话中确认工具列表包含已配置的 MCP Server

安全Important Notes

  • 不要把包含 API Key 的配置文件提交到公开仓库。
  • 建议将配置文件加入 .gitignore。
  • 使用High Risk MCP 前确认权限范围。
