Supported Clients

Installation Command

uvx mcp-server-git

Configuration Example

{
  "mcpServers": {
    "git": {
      "command": "uvx",
      "args": [
        "mcp-server-git",
        "--repository",
        "/path/to/repo"
      ]
    }
  }
}

Permission Scope & Security Notes

只读取 Git 仓库信息，不执行写入操作

  • file-read
