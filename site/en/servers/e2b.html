Installation Command

npx -y @e2b/mcp-server

Configuration Example

{
  "mcpServers": {
    "e2b": {
      "command": "npx",
      "args": [
        "-y",
        "@e2b/mcp-server"
      ],
      "env": {
        "E2B_API_KEY": "<your-e2b-api-key>"
      }
    }
  }
}

Environment variable required：E2B_API_KEY。Do not commit these values to public repositories.
