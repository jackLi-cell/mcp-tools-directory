Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

npx -y @anthropic/mcp-playwright

Configuration Example

{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "-y",
        "@anthropic/mcp-playwright"
      ]
    }
  }
}

Permission Scope & Security Notes

可控制浏览器访问网页并执行操作，注意不要在已登录状态下使用

  • network-access
  • browser-control

Risk Score：5/10

Common Errors

浏览器未安装
运行 npx playwright install chromium 安装浏览器
超时错误
网络慢时增加超时设置
