Installation Command

npx -y exa-mcp-server

Configuration Example

{
  "mcpServers": {
    "exa": {
      "command": "npx",
      "args": [
        "-y",
        "exa-mcp-server"
      ],
      "env": {
        "EXA_API_KEY": "<your-api-key>"
      }
    }
  }
}

Environment variable required：EXA_API_KEY。Do not commit these values to public repositories.
