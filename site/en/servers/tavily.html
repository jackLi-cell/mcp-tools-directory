Installation Command

npx -y tavily-mcp

Configuration Example

{
  "mcpServers": {
    "tavily": {
      "command": "npx",
      "args": [
        "-y",
        "tavily-mcp"
      ],
      "env": {
        "TAVILY_API_KEY": "<your-api-key>"
      }
    }
  }
}

Environment variable required：TAVILY_API_KEY。Do not commit these values to public repositories.
