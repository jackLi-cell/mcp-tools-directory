Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

uvx mcp-server-time --local-timezone Asia/Shanghai

Configuration Example

{
  "mcpServers": {
    "time": {
      "command": "uvx",
      "args": [
        "mcp-server-time",
        "--local-timezone",
        "Asia/Shanghai"
      ]
    }
  }
}
