Medium Risk Category: Design & Product Collaboration Verified: 2026-05-18 状态：部分核验

Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

uvx mcp-atlassian

Configuration Example

{
  "mcpServers": {
    "atlassian": {
      "command": "uvx",
      "args": [
        "mcp-atlassian",
        "--jira-url",
        "https://your-domain.atlassian.net",
        "--jira-token",
        "<your-token>",
        "--confluence-url",
        "https://your-domain.atlassian.net/wiki",
        "--confluence-token",
        "<your-token>"
      ],
      "env": {}
    }
  }
}

Environment variable required：JIRA_URL, JIRA_TOKEN。Do not commit these values to public repositories.

Permission Scope & Security Notes
