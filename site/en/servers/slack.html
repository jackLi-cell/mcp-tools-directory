High Risk Category: Design & Product Collaboration Verified: 2026-05-18 Status: Verified

Supported Clients

  • Claude Desktop
  • Cursor

Installation Command

npx -y @modelcontextprotocol/server-slack

Configuration Example

{
  "mcpServers": {
    "slack": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-slack"
      ],
      "env": {
        "SLACK_BOT_TOKEN": "xoxb-your-token",
        "SLACK_TEAM_ID": "T0123456789"
      }
    }
  }
}

Environment variable required：SLACK_BOT_TOKEN, SLACK_TEAM_ID。Do not commit these values to public repositories.

Permission Scope & Security Notes

可读取和发送 Slack 消息，Required Bot Token，建议限制频道权限

  • network-access
  • credential-required
  • message-send

Risk Score：6/10

Common Errors

确认 Bot 已被邀请到目标频道
