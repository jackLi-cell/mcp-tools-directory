Medium Risk Category: Deploy & Operations Verified: 2026-05-18 状态：部分核验

Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

npx -y @datadog/mcp-server

Configuration Example

{
  "mcpServers": {
    "datadog": {
      "command": "npx",
      "args": [
        "-y",
        "@datadog/mcp-server"
      ],
      "env": {
        "DD_API_KEY": "<your-datadog-api-key>",
        "DD_APP_KEY": "<your-datadog-app-key>"
      }
    }
  }
}

Environment variable required：DD_API_KEY, DD_APP_KEY。Do not commit these values to public repositories.

Permission Scope & Security Notes

主要用于读取监控数据，Required API Key 和 App Key，建议使用只读权限

  • network-access
  • credential-required

Risk Score：3/10

Common Errors

403 Forbidden
确认 API Key 和 App Key 配对正确
找不到指标
确认指标名称正确且在指定时间范围内有数据
