Installation Command

npx -y mcp-obsidian /path/to/vault

Configuration Example

{
  "mcpServers": {
    "obsidian": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-obsidian",
        "/Users/you/ObsidianVault"
      ]
    }
  }
}
