Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

npx -y @brightdata/mcp

Configuration Example

{
  "mcpServers": {
    "bright-data": {
      "command": "npx",
      "args": [
        "-y",
        "@brightdata/mcp"
      ],
      "env": {
        "API_TOKEN": "<your-brightdata-api-token>"
      }
    }
  }
}

Environment variable required：API_TOKEN。Do not commit these values to public repositories.

Permission Scope & Security Notes
