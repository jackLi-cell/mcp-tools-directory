DuckDB 嵌入式分析数据库，支持 SQL 查询、数据导入和 OLAP 分析

Medium Risk Category: Database & Data Query Verified: 2026-05-18 状态：部分核验

Supported Clients

  • Claude Desktop
  • Cursor
  • VS Code / Cline

Installation Command

npx -y mcp-server-duckdb

Configuration Example

{
  "mcpServers": {
    "duckdb": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-server-duckdb",
        "--db-path",
        "/path/to/database.duckdb"
      ]
    }
  }
}

Permission Scope & Security Notes

可读写 DuckDB 数据库文件，建议使用测试数据库

  • database-read
  • database-write
  • file-read

Risk Score：4/10

Common Errors

数据库文件不存在
确认 --db-path 指向有效路径，或省略参数使用内存数据库
文件被锁定
确认没有其他进程正在使用该 DuckDB 文件
