SQLite MCP Server 配置与使用说明

SQLite 数据库操作，支持查询、分析和基本的数据管理

中风险 场景：数据库与数据查询 核验：2026-05-18 状态：已核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

uvx mcp-server-sqlite --db-path /path/to/database.db

配置示例

{
  "mcpServers": {
    "sqlite": {
      "command": "uvx",
      "args": [
        "mcp-server-sqlite",
        "--db-path",
        "/path/to/test.db"
      ]
    }
  }
}

权限范围与安全提示

可读写指定数据库文件，建议使用副本或测试数据库

  • database-read
  • database-write

风险评分：5/10

常见报错

数据库文件不存在
确认 --db-path 指向的文件存在
uvx 命令找不到
需要先安装 uv：curl -LsSf https://astral.sh/uv/install.sh | sh
database is locked
确认没有其他进程正在使用该数据库
