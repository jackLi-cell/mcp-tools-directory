Obsidian MCP Server 配置与使用说明

Obsidian 笔记库读写，支持搜索、创建和编辑笔记

中风险 场景：搜索与知识库 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor

安装命令

npx -y mcp-obsidian /path/to/vault

配置示例

{
  "mcpServers": {
    "obsidian": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-obsidian",
        "/Users/you/ObsidianVault"
      ]
    }
  }
}

权限范围与安全提示

可读写指定 Obsidian 笔记库中的文件

  • file-read
  • file-write

风险评分：3/10

常见报错

Vault 路径不存在
确认路径指向有效的 Obsidian Vault 目录
文件编码错误
确认笔记文件为 UTF-8 编码
