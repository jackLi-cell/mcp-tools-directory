ArXiv MCP Server 配置与使用说明

📦
安装方式uvx
⚠️
风险等级低风险
🔑
API Key不需要
💻
客户端3 个支持

ArXiv 学术论文搜索，支持按关键词、作者和分类检索论文摘要

低风险 场景：搜索与知识库 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

uvx arxiv-mcp-server

配置示例

{
  "mcpServers": {
    "arxiv": {
      "command": "uvx",
      "args": [
        "arxiv-mcp-server"
      ]
    }
  }
}

权限范围与安全提示

只查询公开的 ArXiv 论文数据，不需要 API Key，风险极低

  • network-access

风险评分：1/10

常见报错

网络超时
ArXiv API 有时响应较慢，增加超时时间或重试
uvx 找不到
安装 uv：curl -LsSf https://astral.sh/uv/install.sh | sh
