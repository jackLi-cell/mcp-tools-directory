Tavily Search MCP Server 配置与使用说明

Tavily AI 搜索引擎，专为 AI Agent 优化的搜索 API

低风险 场景：搜索与知识库 核验：2026-05-18 状态：已核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

npx -y tavily-mcp

配置示例

{
  "mcpServers": {
    "tavily": {
      "command": "npx",
      "args": [
        "-y",
        "tavily-mcp"
      ],
      "env": {
        "TAVILY_API_KEY": "<your-api-key>"
      }
    }
  }
}

需要设置环境变量：TAVILY_API_KEY。不要把这些值提交到公开仓库。

权限范围与安全提示

只进行搜索查询，不修改任何数据，免费版每月 1000 次

  • network-access
  • credential-required

风险评分：2/10
