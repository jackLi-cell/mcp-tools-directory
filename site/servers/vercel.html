Vercel MCP Server 配置与使用说明

Vercel 部署平台管理，支持项目、部署和域名操作

高风险 场景：部署与运维 核验：2026-05-18 状态：已核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

npx -y @vercel/mcp

配置示例

{
  "mcpServers": {
    "vercel": {
      "command": "npx",
      "args": [
        "-y",
        "@vercel/mcp"
      ],
      "env": {
        "VERCEL_TOKEN": "<your-token>"
      }
    }
  }
}

需要设置环境变量：VERCEL_TOKEN。不要把这些值提交到公开仓库。

权限范围与安全提示

可管理 Vercel 项目和部署，建议使用受限 Token

  • network-access
  • credential-required
  • infrastructure-write

风险评分：6/10

常见报错

认证失败
在 Vercel Settings > Tokens 生成 Token
项目未找到
确认 Token 有该项目的访问权限
