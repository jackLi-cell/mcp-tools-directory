MySQL MCP Server 配置与使用说明

MySQL 数据库连接，支持查询、表结构查看和基本数据操作

高风险 场景：数据库与数据查询 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor

安装命令

npx -y @benborla29/mcp-server-mysql

配置示例

{
  "mcpServers": {
    "mysql": {
      "command": "npx",
      "args": [
        "-y",
        "@benborla29/mcp-server-mysql"
      ],
      "env": {
        "MYSQL_HOST": "localhost",
        "MYSQL_PORT": "3306",
        "MYSQL_USER": "root",
        "MYSQL_PASSWORD": "<password>",
        "MYSQL_DATABASE": "mydb"
      }
    }
  }
}

需要设置环境变量：MYSQL_HOST, MYSQL_USER, MYSQL_PASSWORD, MYSQL_DATABASE。不要把这些值提交到公开仓库。

权限范围与安全提示

可读写 MySQL 数据库，强烈建议使用只读账号和测试数据库

  • database-read
  • database-write
  • network-access
  • credential-required

风险评分：6/10

常见报错

连接被拒绝
确认 MySQL 服务运行中且允许连接
Access denied
检查用户名、密码和数据库权限
