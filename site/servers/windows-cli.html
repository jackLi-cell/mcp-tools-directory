安装命令

npx -y win-cli-mcp-server

配置示例

{
  "mcpServers": {
    "windows-cli": {
      "command": "npx",
      "args": [
        "-y",
        "win-cli-mcp-server"
      ]
    }
  }
}

权限范围与安全提示

可执行任意 Windows 命令（PowerShell/CMD），风险极高，仅在完全信任的环境中使用

  • command-execution
  • file-read
  • file-write
  • network-access

风险评分：9/10

常见报错

命令执行超时
长时间运行的命令可能超时，尝试拆分为多个短命令
权限不足
部分命令需要管理员权限，以管理员身份运行客户端
