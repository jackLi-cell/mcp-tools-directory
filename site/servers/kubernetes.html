Kubernetes MCP Server 配置与使用说明

Kubernetes 集群管理，支持查看 Pod、Deployment、Service 等资源状态

高风险 场景：部署与运维 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

npx -y mcp-server-kubernetes

配置示例

{
  "mcpServers": {
    "kubernetes": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-server-kubernetes"
      ]
    }
  }
}

权限范围与安全提示

使用本地 kubeconfig 访问 K8s 集群，可查看和管理集群资源，切勿连接生产集群

  • network-access
  • infrastructure-write
  • credential-required

风险评分：7/10

常见报错

kubeconfig 未找到
确认 ~/.kube/config 文件存在且配置正确
连接被拒绝
确认集群可达且 kubeconfig 中的 context 正确
