Time MCP Server 配置与使用说明

时间和时区工具，支持获取当前时间和时区转换

低风险 场景：文件与本地系统 核验：2026-05-18 状态：已核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

uvx mcp-server-time --local-timezone Asia/Shanghai

配置示例

{
  "mcpServers": {
    "time": {
      "command": "uvx",
      "args": [
        "mcp-server-time",
        "--local-timezone",
        "Asia/Shanghai"
      ]
    }
  }
}

权限范围与安全提示

只提供时间信息，不访问文件或网络

    风险评分：0/10

    常见报错

    时区名称无效
    使用 IANA 时区名称如 Asia/Shanghai、America/New_York
