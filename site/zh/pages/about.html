关于本站

本站是面向中文用户的 AI Agent 与 MCP Server 工具目录，帮助开发者按使用场景、客户端、安装方式和权限范围筛选工具，并提供配置步骤、常见错误排查和安全检查清单。

内容来源

工具信息来自官方文档、GitHub 仓库、npm/PyPI 包页面和客户端官方文档。每个工具条目标注最近核验日期，超过 60 天未核验的标记为"待复核"。

本站不做什么

  • 不托管用户密钥或配置文件
  • 不代理用户调用任何 API
  • 不提供远程执行服务
  • 不保证第三方工具的安全性和可用性
  • 不提供绕过平台限制或攻击性自动化教程
