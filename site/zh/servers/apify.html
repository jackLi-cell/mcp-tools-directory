Apify 爬虫平台集成，提供 4000+ 预构建爬虫工具（Actors）

中风险 场景：浏览器与网页自动化 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

npx -y @apify/actors-mcp-server

配置示例

{
  "mcpServers": {
    "apify": {
      "command": "npx",
      "args": [
        "-y",
        "@apify/actors-mcp-server"
      ],
      "env": {
        "APIFY_TOKEN": "<your-apify-token>"
      }
    }
  }
}

需要设置环境变量：APIFY_TOKEN。不要把这些值提交到公开仓库。

权限范围与安全提示

通过 Apify 平台运行爬虫任务，需要 API Token，会消耗平台计算资源

  • network-access
  • credential-required

风险评分：4/10

常见报错

Token 无效
在 Apify Console > Settings > Integrations 中获取 Token
Actor 不存在
确认 Actor ID 正确，可在 Apify Store 中搜索
