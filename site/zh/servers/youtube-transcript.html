YouTube Transcript MCP Server 配置与使用说明

📦
安装方式npx
⚠️
风险等级低风险
🔑
API Key不需要
💻
客户端3 个支持

YouTube 视频字幕下载，支持获取视频的完整文字记录

低风险 场景：搜索与知识库 核验：2026-05-18 状态：部分核验

支持客户端

  • Claude Desktop
  • Cursor
  • VS Code / Cline

安装命令

npx -y mcp-server-youtube-transcript

配置示例

{
  "mcpServers": {
    "youtube-transcript": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-server-youtube-transcript"
      ]
    }
  }
}

权限范围与安全提示

只读取 YouTube 公开视频的字幕数据，不需要 API Key

  • network-access

风险评分：1/10

常见报错

字幕不可用
该视频可能没有字幕或字幕被禁用
视频不存在
确认视频 URL 或 ID 正确且视频未被删除
