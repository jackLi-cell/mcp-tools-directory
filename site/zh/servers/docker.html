Docker 容器管理，支持列出容器、查看日志、执行命令和管理镜像

高风险 场景：部署与运维 核验：2026-05-18 状态：已核验

支持客户端

  • Claude Desktop
  • Cursor

安装命令

npx -y @docker/mcp-server-docker

配置示例

{
  "mcpServers": {
    "docker": {
      "command": "npx",
      "args": [
        "-y",
        "@docker/mcp-server-docker"
      ]
    }
  }
}

权限范围与安全提示

可管理 Docker 容器，包括启停和执行命令，需要 Docker 运行中

  • command-execution
  • infrastructure-write

风险评分：7/10

常见报错

Docker daemon 未运行
启动 Docker Desktop 或 dockerd 服务
permission denied
确认当前用户在 docker 组中
